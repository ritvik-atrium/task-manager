# **App Name**: TaskNest

## Core Features:

- Category & Task Creation: Allow users to create and manage task categories, and add new tasks within them. Each task can also be the parent of multiple nested subtasks, with no limit to nesting depth.
- Hierarchical Task Display: Visualize tasks and their nested subtasks in an intuitive, indented tree-like structure within each category for easy management and overview.
- Task Status Management: Users can mark tasks and subtasks as complete or uncomplete, providing a clear indication of progress.
- Task Reordering: Enable drag-and-drop functionality to reorder tasks and subtasks within their respective categories and parent tasks.
- Local JSON Data Storage: Automatically save all task data, including categories and nested structures, to a local JSON file, and load it upon application launch.
- Task & Subtask Editing: Users can edit existing task and subtask details, including descriptions and category assignments.
- Subtask Breakdown Tool: An AI-powered tool that suggests potential subtasks for a given primary task, assisting users in breaking down complex projects into manageable steps.

## Style Guidelines:

- A focused, productivity-oriented light theme. The primary color is a deep, clear blue (#1F83A7) symbolizing clarity and stability. The background is a very light, almost white cool gray (#EDF2F4), providing a clean canvas. An accent color, a refreshing greenish-cyan (#3DBA8C), highlights interactive elements and actions.
- The sans-serif font 'Inter' is recommended for both headlines and body text. Its modern, objective design ensures high legibility across task descriptions and hierarchical structures, contributing to an organized and efficient user experience.
- Use a set of simple, clean, and intuitive outline icons for actions such as adding tasks, marking completion, editing, and expanding/collapsing subtasks, ensuring clarity without visual clutter.
- The layout should feature a clear sidebar for categories and a main content area for tasks. Tasks and subtasks will be presented in an indented, hierarchical list for easy comprehension of their relationships. Ample white space will be used to reduce visual noise.
- Subtle, fluid animations should be implemented for opening/closing subtask lists, marking tasks complete, and reordering items, providing a responsive and polished feel without being distracting.
